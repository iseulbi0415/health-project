# 🏃‍♀️ 건강 케어 & 러닝 트래커 웹 프로젝트

역류성 식도염을 안전하게 케어하고, 무리 없는 유산소(러닝) 습관을 만들기 위한 개인 맞춤형 웹 서비스입니다.
복잡한 외부 API 연동 없이, 프론트엔드부터 백엔드까지 전체 개발 사이클을 직접 완주하는 것을 목표로 학습하며 만들고 있습니다.

## 핵심 기능
- 🔥 **소화 안전 타이머** — 식사 완료 시, 먹은 음식 기준으로 안전하게 누울 수 있는 시간을 계산해 카운트다운
- 🥗 **식단 & 칼로리 기록** — 자주 먹는 음식 빠른 추가 + 오늘 총 섭취 칼로리 자동 합산
- 👟 **러닝 기록 일지** — 거리, 시간, 페이스 등 기록 관리
- 📋 **BMR 기반 목표 칼로리 계산기**
- 📝 **컨디션 메모장**

## 기술 스택
- **Frontend**: HTML, CSS, JavaScript
- **Backend (예정)**: Java, Spring Boot, MySQL
- **Tools**: VS Code, GitHub
